42 /*
43  * These macros enables features of SDHC driver:
44  *
45  * BSP_FSL_SDHC_USING_IRQ
46  * - Enables IRQ on Initialization
47  *
48  * BSP_FSL_SDHC_ENABLE_ADMA1
49  * - Enables the ADMA1 feature. As a result of the alignment limitation of the ADMA1,
50  * it falls back to PIO if the data size/address does not align.
51  *
52  * BSP_FSL_SDHC_ENABLE_AUTOCMD12
53  * - Enables sending cmd12 automatically for multiple block R/W access.
54  *
55  * BSP_FSL_SDHC_USING_DYNALLOC
56  * - Enables dynamic allocate memory in SDHC/SDCARD driver
57  *
58  */
