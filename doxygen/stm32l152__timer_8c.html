HAL_StatusTypeDef HAL_InitTick ( uint32_t  TickPriority)

This function configures the TIM11 as a time base source. The time source is configured to have 1ms time base with a dedicated Tick interrupt priority.

Note
This function is called automatically at the beginning of program after reset by HAL_Init() or at any time when clock is configured, by HAL_RCC_ClockConfig().
