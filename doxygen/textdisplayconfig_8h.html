#define TEXTDISPLAY_DEVICE_0_LINES   (DISPLAY0_HEIGHT / FONT_HEIGHT)

Determine the number of lines and columns of the text display devices. These constants are used for static memory allocation in the textdisplay device driver.

Please make sure that the combined selection of font, lines and columns fits inside the DISPLAY geometry.

#define TEXTDISPLAY_DEVICES_MAX   (1)

Maximum number of text display devices the display module is configured to support. This number may be increased if the system includes more than one display device. However, the number should be kept low in order to save memory.
